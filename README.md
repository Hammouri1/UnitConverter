# Unit Converter 📏

A simple, intuitive Android application built with Kotlin and Jetpack Compose that allows users to easily convert between different units of length. 

## 📸 Screenshots

![Unit Converter App](Image.png)

## ✨ Features

* **Real-Time Conversion:** The app calculates the converted value instantly as you type or change units.
* **Quick Swap:** Features a handy "⇄" button to instantly swap your input and output units without having to reselect them from the dropdowns.
* **Modern UI:** Built entirely using Android's modern declarative UI toolkit, Jetpack Compose, and Material Design 3 components.
* **Clean Architecture:** Utilizes Compose state management (`remember`, `mutableStateOf`) for a smooth, reactive user experience.

## 📐 Supported Units

Currently, the app supports conversion between the following length measurements:
* Meters
* Centimeters
* Millimeters
* Feet

## 🛠️ Tech Stack

* **Language:** [Kotlin](https://kotlinlang.org/)
* **UI Toolkit:** [Jetpack Compose](https://developer.android.com/jetpack/compose)
* **Design:** Material Design 3
* **IDE:** Android Studio
